# acdecogarch

Implements quasi-maximum-likelihood estimation of the ECOGARCH(1,1)
model on irregularly spaced data.

## Installation

You can install the package from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pkg_install("hgstp/acdecogarch")
```
